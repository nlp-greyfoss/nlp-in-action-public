# Data
The data file is constructed with two sentences separated by a tab (`\t`), and there is a numerical value indicating whether they are similar.

`data/train.txt`:
```
Hello	Hi	1
Nice to see you.	Nice	0
```

# Train

Change arguments in `train.sh`.

```sh
sh train.sh
```
